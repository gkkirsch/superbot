Getting Started

Clone the repo, run setup, and you're in.

Prerequisites

Superbot is built on Claude Code, Anthropic's CLI for Claude. The setup script will install it for you if you don't have it already.

You'll need an Anthropic plan that supports Claude Code. We recommend a Max plan — superbot runs background workers, heartbeat tasks, and Slack alongside your main session, which uses significant capacity.

1 Clone the Repo 

git clone https://github.com/gkkirsch/superbot.git ~/superbot
cd ~/superbot && npm install

2 Run Setup 

bash ~/superbot/scripts/setup.sh

This scaffolds all config files, installs skills, sets up your shell, and launches straight into onboarding. The onboarding walks you through naming your bot, defining its personality, and telling it about yourself. When it's done, superbot restarts with your new config loaded and you're ready to go.
