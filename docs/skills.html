Skills

Extend superbot's capabilities with installable skills.

What Are Skills?

Skills are packaged capabilities that teach Claude how to do specific things — interact with Google Workspace, deploy to Cloudflare, manage Slack, and more. Each skill is a directory with a SKILL.md file that contains instructions and reference documentation.

Built-in Skills

Superbot ships with 3 built-in skills in the skills/ directory:

Skill Description
superbot Context manager — memory, identity, user profile. Commands: remember, forget, identity, user, status.
slack Slack messaging — formatting, best practices, and how the bot works. Teaches superbot to write good Slack messages.
gog Google Workspace CLI — Gmail, Calendar, Drive, Contacts, Sheets, Docs. Requires OAuth setup.

Managing Skills

/superbot:skills list                # Show available and installed skills
/superbot:skills install gog         # Install a built-in skill
/superbot:skills install slack       # Install from the registry
/superbot:skills remove gog          # Remove an installed skill
/superbot:skills search calendar     # Search the registry

Installing a built-in skill

Built-in skills are copied from skills/<name>/ in the superbot repo to ~/.claude/skills/<name>/:

cp -r ~/superbot/skills/gog ~/.claude/skills/gog

Some skills have prerequisites. The gog skill, for example, requires a separate OAuth setup for Google Workspace access.

Installing from the registry

Skills not in the built-in catalog are fetched from the skills.sh registry:

npx skills add anthropics/skills@slack-gif-creator -g -a claude-code -y

The format is owner/repo@skill-name.

Searching the Registry

The skills registry API lets you discover skills:

GET https://skills.sh/api/search?q=<query>&limit=50

Example response:

{
  "query": "slack",
  "searchType": "fuzzy",
  "skills": [
    {
      "id": "anthropics/skills/slack-gif-creator",
      "skillId": "slack-gif-creator",
      "name": "slack-gif-creator",
      "installs": 3133,
      "source": "anthropics/skills"
    }
  ],
  "count": 3
}

The id field maps to the install command: anthropics/skills/slack-gif-creator becomes anthropics/skills@slack-gif-creator.

Prefer skills with higher install counts and from reputable sources (anthropics/, cloudflare/, vercel-labs/, firecrawl/).

Skill File Structure

Each skill is a directory with at minimum a SKILL.md file:

skills/
+-- superbot/
|   +-- SKILL.md
+-- slack/
|   +-- SKILL.md
+-- gog/
    +-- SKILL.md

The SKILL.md contains the skill's instructions, commands, and reference documentation that Claude reads when the skill is invoked.
